# Proton Plus

<figure><img src="../../.gitbook/assets/protonplus_480141_full.png" alt=""><figcaption></figcaption></figure>

Установить ProtonPlus в ALT Linux

```bash
sudo apt-get install protonplus
```

через EPM

```bash
epmi protonplus
```

Flatpak

```bash
flatpak install flathub com.vysp3r.ProtonPlus
```
