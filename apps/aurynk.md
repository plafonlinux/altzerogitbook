### Подготовка устройства:

01\. Включаем **Режим разработчика**, если ещё не включали, обычно в меню "О телефоне" нужно потыкать много (7-10 раз) на Номер сборки (Build Number) до получения уведомления.

02\. Включаем отладку по USB и WiFi (я включил оба варианта)

В секции "Отладка" включаем 3 пункта:

&#x20;                  \- Отладка по USB\
&#x20;                  \- Беспроводная отладка\
&#x20;                  \- Отладка по USB (Настройки безопасности)

{% hint style="danger" %}
Чтобы включить данный пункт Xiaomi заставляет залогинится в свой Xiaomi аккаунт и в смартфоне должно быть наличие SIM-карты. Без данного пункта вы не сможете взаимодействовать с вашим смартфоном, получите только отображение на экране. SIМ-карту я вернул обратно в свой основной смартфон, но я не уверен что этот пункт не отвалится после перезагрузки.
{% endhint %}

{% hint style="warning" %}
На всякий случай я отключал ещё и пункт "Оптимизация MIUI"\
\
Как это сделать?

* Прокрутите в самый низ до раздела «Восстановить значения по умолчанию».
* Нажмите на этот пункт много раз подряд, пока ниже не появится скрытый переключатель «Включить оптимизацию MIUI».
{% endhint %}

### Вариант А (Ручной через терминал)

03\. Подключаем смартфон через adb

{% hint style="success" %}
В моём случае на ALT Linux p11 android-tools уже был установлен.
{% endhint %}

Нажимаем на пункт "Беспроводная отладка" и проваливаем в подменю

В случае с ручным подключение через терминал тапаем на пункт "Подключение устройств через код подключение"

<div><figure><img src="../.gitbook/assets/Снимок экрана от 2026-01-03 15-28-40.png" alt=""><figcaption></figcaption></figure> <figure><img src="../.gitbook/assets/Снимок экрана от 2026-01-03 15-28-30.png" alt=""><figcaption></figcaption></figure></div>

```bash
adb pair 192.168.2.72:37439
```

Жмём ввод и терминал запросит код подключение: Вводим 6-значный код как на скриншоте выше.

После того как мы успешно спарили устройства, можно подключиться к нему (Вводим адрес и порт как на втором скриншоте):

```bash
adb connect 192.168.2.72:41433
```

{% hint style="success" %}
Если всё сделали правильно терминал напишет: connected ....
{% endhint %}

Устанавливаем утилиту для mirrorcast

```bash
epmi scrcpy
```

После чего запускаем в терминале и если всё сделали правильно видим экран нашего смартфона:

### Вариант Б (для ленивцев как я)

Для всего этого дела на Flathub есть клиент который значительно упрощает клиент подключение смартфона:

01\. Устанавливаем сам клиент Aurynk

```bash
flatpak install flathub io.github.IshuSinghSE.aurynk
```

02\. Заходим в клиент и нажимаем на + появится Qr код для подключения

<figure><img src="../.gitbook/assets/Снимок экрана от 2026-01-03 12-31-08.png" alt=""><figcaption></figcaption></figure>

Заходим в подменю "Беспроводная отладка" и выбираем 3-й пункт "Подключение устройств через QR-код" и сканируем наш QR, после чего сматрфон успешно подключается и всё готово.

<figure><img src="../.gitbook/assets/Снимок экрана от 2026-01-03 12-31-33.png" alt=""><figcaption></figcaption></figure>



### Если adb завис

Если вы немного пофейлили с подключениями как и я, то adb может подвиснуть, перезапускаем сервис и подключаемся заново.

```
adb kill-server
```

```
adb start-server
```
